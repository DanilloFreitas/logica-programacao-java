# Lógica de Programação e POO com Java ☕

Este repositório foi criado para documentar a minha evolução nos estudos de Java, focado em lógica de programação e nos pilares da **Programação Orientada a Objetos (POO)**.

## 🚀 Projetos Atuais

### 1. Calculadora POO
Uma calculadora funcional desenvolvida para validar conceitos de encapsulamento e separação de responsabilidades.

#### Conceitos Aplicados:
- **Encapsulamento:** Atributos privados (`private`) acessados via getters e setters.
- **Lógica de Negócio:** Métodos dedicados para operações matemáticas, mantendo a classe Main apenas para interação com o usuário.
- **Tratamento de Exceções:** Verificação lógica para evitar erros comuns, como a divisão por zero.
